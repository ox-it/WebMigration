Service Level Agreements

A Service Level Agreement (SLA) formally defines the level of service that a customer can expect, and that the service provider is committed to providing and maintaining.

In our case, it is an understanding between the end-user and BSP, with respect to a given system that BSP supports. An SLA covers issues such as system performance, security, availability, down-time, problem resolution and response times.

The table below shows which systems are covered by which SLA.

For more information on SLAs, please contact the appropriate Helpdesk for that system.
