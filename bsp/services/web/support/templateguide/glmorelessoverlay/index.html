GL More/Less - Overlay gallery

The ‘GL More/Less - Overlay gallery’ template is very similar to 'Gl More/Less' except that it allows more characters in the text fields and allows a photo gallery of up to five images inside the expandable content.  These images are presented as thumbnails and expand to a larger image when clicked.

Examples of this can be found on the Access Guide http://www.admin.ox.ac.uk/access/libraries/bodleianlawlibrary/
